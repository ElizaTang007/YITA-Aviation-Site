API 文档

完整的API参考文档和示例

设备管理

GET /api/v1/devices 获取设备列表
POST /api/v1/devices 注册新设备
PUT /api/v1/devices/{id} 更新设备信息

任务控制

POST /api/v1/missions 创建任务
GET /api/v1/missions/{id} 获取任务状态
PUT /api/v1/missions/{id}/control 控制任务执行

实时数据

GET /api/v1/telemetry/{device_id} 获取遥测数据
GET /api/v1/video/{device_id}/stream 获取视频流

示例代码

Python - 创建巡检任务 
import yita_sdk

# 初始化客户端
client = yita_sdk.Client(api_key="your_api_key")

# 创建巡检任务
mission = client.missions.create(
    device_id="drone-001",
    mission_type="inspection",
    waypoints=[
        {"lat": 39.9042, "lng": 116.4074, "alt": 100},
        {"lat": 39.9045, "lng": 116.4078, "alt": 120},
        {"lat": 39.9040, "lng": 116.4080, "alt": 100}
    ],
    parameters={
        "inspection_mode": "auto",
        "camera_resolution": "4k",
        "return_home": True
    }
)

print(f"任务已创建: {mission.id}")

# 监控任务状态
while mission.status != "completed":
    mission = client.missions.get(mission.id)
    print(f"任务状态: {mission.status}")
    time.sleep(5)
